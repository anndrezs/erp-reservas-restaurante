# ERP de Reservas de Restaurante

Sistema completo de gerenciamento de reservas para restaurantes, desenvolvido com:

- **Node.js** + **Express**
- **TypeScript**
- **MongoDB** com Mongoose
- **JWT Auth + Controle de Acesso por Cargo**
- **Envio de E-mail com Nodemailer**
